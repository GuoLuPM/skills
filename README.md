# 如何使用

- 通用规则：你直接点名 skill 名称，或你的需求描述命中它的触发条件，Codex 就会用它。

- `manual-frontend-debugging`
  - 触发：你能自己复现前端 bug，希望 Codex 先挂短日志/临时探针，你操作一遍，再让它看日志定位。
  - 你可以直接说：`你先挂短日志，我来复现，你看日志。`
